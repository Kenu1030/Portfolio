# Portfolio (HTML + CSS)

A simple 1-page IT-focused portfolio site built with plain HTML and CSS.

## How to run

Option A (simplest):
- Open `index.html` in your browser.

Option B (recommended while editing):
- Start a local web server so links/paths behave consistently.

PowerShell (Python installed):
- `python -m http.server 5500`
- Open `http://localhost:5500`

## What to edit first

1. `index.html`
   - Update any remaining placeholders like `[System/Website Name]`.
   - Replace the 3 project cards with your real project names and links.
   - (Optional) Add LinkedIn/GitHub links if you want them shown.

2. `css/styles.css`
   - Adjust spacing / font sizes if you want.
   - Keep the CSS variables in `:root` as your design controls.
