# Portfolio

A personal portfolio website showcasing my web development projects, technical skills, and academic background, designed to demonstrate my ability to build responsive and user-friendly web applications.

---

## 📖 Overview

This portfolio serves as a central hub where visitors can learn about me, explore the projects I have built, and get in touch. It highlights my journey as a developer, the technologies I work with, and my passion for creating clean, functional web experiences.

---

## ✨ Features

- **About Me** – A brief introduction covering my background, interests, and goals as a developer.
- **Projects** – A curated showcase of personal and academic projects with descriptions and links.
- **Skills** – A visual summary of the programming languages, frameworks, and tools I use.
- **Contact** – Easy ways to reach me via email or social/professional profiles.
- **Responsive Design** – Fully optimized for desktop, tablet, and mobile screen sizes.

---

## 🛠️ Technologies Used

- **HTML5** – Semantic markup and page structure
- **CSS3** – Styling, animations, and responsive layout
- **JavaScript** – Interactivity and dynamic content
- **Git & GitHub** – Version control and project hosting

---

## 🚀 Getting Started

To run this project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Kenu1030/Portfolio.git
   ```

2. **Navigate into the project directory**
   ```bash
   cd Portfolio
   ```

3. **Open `index.html` in your browser**
   ```bash
   open index.html
   ```
   Or simply double-click the file in your file explorer.

No build tools or dependencies are required – the project runs entirely in the browser.
